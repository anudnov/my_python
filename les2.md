Задано натуральное число n. Делится ли оно одновременно на a и на b ?
Входные данные -
Три натуральных числа n, a, b, не больше 10 в 9 степени.

Выходные данные:
Выведите YES ессли n делится одновременно на a и на b.
Выведите NO иначе.

```
n = int(input('n= ' ))
a = int(input('a= ' ))
b = int(input('b= ' ))

if n % a == 0 and n % b == 0:
    print('YES')
else:
    print('NO')
```
